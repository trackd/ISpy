# Get-DecompilerVersion

## SYNOPSIS

Gets the version of the ICSharpCode.Decompiler engine used by the ISpy module.

## SYNTAX

```powershell
Get-DecompilerVersion [<CommonParameters>]
```

## DESCRIPTION

The `Get-DecompilerVersion` cmdlet returns the version string of the ICSharpCode.Decompiler library used by the module. Useful for troubleshooting and compatibility checking.

## OUTPUTS

### System.String

Returns the decompiler version string.
